### Introduction
App for offline speech-to-text recognition.
* By holding the microphone button, you speak and the text appears on the screen in large font.
* Double-tap to copy the text.
* Font size can be adjusted.
* Background and text color can be customized.

**The app requires access to the microphone to function.**

### Language Support
The app uses the offline **VOSK** library for the Turkish language.

By following the instructions in the app, you can replace the VOSK library and choose another language from the available list:
[https://alphacephei.com/vosk/models](https://alphacephei.com/vosk/models)
